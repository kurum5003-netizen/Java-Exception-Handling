# Java-Exception-Handling
Bu repository, Java Exception Handling (İstisna Yönetimi) konusunu öğrenmek ve pekiştirmek amacıyla hazırlanmış örnek kodları içermektedir.
Amaç, Java’da hata türlerini, hata yakalama mekanizmalarını ve doğru exception yönetimi yaklaşımlarını uygulamalı olarak göstermektir.
